# Описание проекта
## Это приложение реализует две версии игры Крестики нолики

# Классическая игра — стандартное поле 3×3
Мега крестики-нолики — поле из девяти малых досок 3×3, расположенных в одной большой доске 3×3
Игра поддерживает режимы
Игра с другом — поочерёдные ходы двух игроков
Игра с ботом — один игрок против умного компьютерного соперник

# Правила игры:
## Классические крестики-нолики
Два игрока по очереди ставят «X» и «O» на поле 3×3
Побеждает тот, кто первым выстроит три своих символа в линию (горизонталь, вертикаль, диагональ)
Если поле заполнено, а победителя нет — ничья

## Мега крестики-нолики
Поле состоит из 9 малых досок (всего 81 клетка)
После того как вы сделали ход в клетке малой доски, ваш оппонент должен ходить в ту малую доску, которая соответствует позиции вашей клетки
Например: если вы походили в центральную клетку малой доски, следующий ход должен быть на центральной малой доске
Если малая доска заполнена или уже выиграна — следующий игрок может ходить на любую доступную доску
Побеждает тот, кто первым выстроит три выигранные малые доски в линию на глобальной доске
Логика бота
В классической игре:
Бот ищет выигрышный ход для себя
Если нет — ищет ход, блокирующий победу игрока
Если нет — отдаёт приоритет:
Центру (1,1)
Углам (0,0), (0,2), (2,0), (2,2)
Сторонам
Выбирает случайный ход из доступных
В мега-игре:
Аналогично проверяет выигрышные и блокирующие ходы в пределах текущей малой доски
Если ходить можно на любую доску — выбирает случайную доступную
При выборе клетки отдаёт приоритет:
Центральным клеткам
Угловым клеткам
